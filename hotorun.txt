To download the code, 
git clone https://github.com/IradNuriel/Final-Project-Final.git

To run the code, you will need to have a windows 10 machine,

SEAL-Python(link to the library with a tutorial to douwnload: https://github.com/Huelse/SEAL-Python)

smtplib
base64
keras
tensorflow
keras_vggface
numpy
dlib
skimage
sklearn
